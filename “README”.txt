## Environment Settings
- Python: '3.5'
- Numpy: '1.18'
- Pandas: '1.0'

## Comparison methods
- Bayesian Probabilistic Matrix Factorization (BPMF) 
- Regularized SVD (RSVD)
- Matrix Factorization using Alternating Least Squares (ALS)
- Landmarks Support Vector Regression (SVR) 
- Collaborative Filtering to Supervised Learning (COFILS)

## Datasets
We provide a divided datasets: MovieLens 100k (trainset_100k.csv and testset_100k.csv).

**trainset_100k:**
- Train file.
- Each Line is a training instance: userID\t itemID\t rating\t

**testset_100k:**
- Test file (positive instances).
- Each Line is a testing instance: userID\t itemID\t rating\t
